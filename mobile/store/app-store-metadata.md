### Age rating guidance
Recommended rating: **4+ (Everyone)**. The app is built for secondary-school students (minors) and contains no violence, mature themes, or objectionable material. When completing Apple's age-rating questionnaire, note these considerations:
- **AI tutor (Sandra):** an in-app AI chat assistant scoped to academic tutoring. It is not open-ended social chat, but reviewers may ask about generated content — answer "None/Infrequent" for mature content.
- **Embedded video:** lessons stream from YouTube (privacy-enhanced `youtube-nocookie.com`). This is curated educational content, not unrestricted web browsing, so "Unrestricted Web Access" should be **No**.
- **Leaderboard:** users can set a public display alias (no free-form profile text beyond a name, optional school/city). No direct messaging between users.
- **No ads, no in-app purchases, no gambling.** Keep "Made for Kids" **off** in App Store Connect (account creation / online features present), but 4+ is appropriate.
